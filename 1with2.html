一元二次方程求根公式推导过程

1. 从一般形式的一元二次方程开始：

ax2 + bx + c = 0

2. 将方程两边同时除以a：

x2 + (b/a)x + c/a = 0

3. 移项得到：

x2 + (b/a)x = -c/a

4. 在方程两边加上一次项系数的一半的平方：

x2 + (b/a)x + (b/2a)2 = (b/2a)2 - c/a

5. 配方得到：

(x + b/2a)2 = (b2 - 4ac) / 4a2

6. 开平方得到：

x + b/2a = ±√(b2 - 4ac) / 2a

7. 最终得到求根公式：

x = [-b ± √(b2 - 4ac)] / 2a
